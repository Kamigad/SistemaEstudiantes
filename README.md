Aplicación desarrollada en Java que permite gestionar estudiantes mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).
Los datos se almacenan en una base de datos MySQL utilizando conexión directa con JDBC.

Tecnologías utilizadas
Java
MySQL
JDBC (Java Database Connectivity)
IntelliJ IDEA

Funcionalidades

La aplicación permite:

✔ Registrar nuevos estudiantes
✔ Listar todos los estudiantes
✔ Buscar estudiantes por ID
✔ Actualizar información
✔ Eliminar estudiantes
✔ Persistencia en base de datos MySQL

Arquitectura utilizada

El proyecto sigue una estructura básica en capas:

Modelo → Representa la entidad Estudiante
DAO (Datos) → Acceso a la base de datos
Servicio → Lógica de negocio
Main → Punto de entrada de la aplicación

Ejemplo de uso
Crear un estudiante
Consultar lista de estudiantes
Modificar datos
Eliminar registros
